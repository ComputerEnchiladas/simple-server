# simple-server

## Dependencies

`node -v` >= 6.9.1

`npm -v` >= 3.3.10

## Setup
### Clone Repo

`git clone https://github.com/ComputerEnchiladas/simple-server.git`

### Install Dependencies

`npm install`

### Run Server

`node server/index`

### Browser Preview

Open browser and go to http://127.0.0.1:6085 or custom IP:PORT
